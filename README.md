# ConexyTask 

**ConexyTask** — это минималистичное, но функциональное веб-приложение для управления задачами (To-Do лист) со встроенным таймером. Проект разработан как монолит, состоящий из бэкенда на ASP.NET Core Web API и одностраничного фронтенда (SPA) на чистом JavaScript. Особенность проекта — гибридное хранение данных: основная информация о задачах сохраняется в PostgreSQL, а расширенные параметры (приоритеты, таймеры) — в `localStorage` браузера, что позволяет приложению работать даже при недоступности сервера.
Можно попробовать по ссылке https://conexytask.onrender.com

## 🚀 Ключевые возможности

*   **Управление задачами:** Создание, редактирование и удаление задач.
*   **Система приоритетов:** Автоматический расчет приоритета задачи (Красный 🔴, Желтый 🟡, Зеленый 🟢) на основе сложности и срочности (матрица Эйзенхауэра).
*   **Гибкие настройки:** Установка сложности, срочности, дедлайна и типа повторения (ежедневно, еженедельно и т.д.).
*   **Умные повторения:** При выполнении повторяющейся задачи автоматически создается её копия со сдвигом даты.
*   **Встроенный таймер:** Для каждой задачи можно запустить таймер обратного отсчета с возможностью перехода в полноэкранный режим.
*   **Автономный режим (Fallback):** Если сервер недоступен, все операции выполняются в `localStorage`.
*   **Адаптивный интерфейс:** Стильный дизайн в стиле "стекло" (glassmorphism) корректно отображается на всех устройствах.
*   **Чистая архитектура:** Бэкенд разделен на четкие слои (Controller, Service, Repository), что облегчает поддержку и масштабирование.
